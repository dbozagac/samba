# Samba Web (Next.js)

## Run locally
```bash
cp .env.example .env.local
npm install
npm run dev
```

Default URL: `http://localhost:3000`

## Required env vars
- `NEXT_PUBLIC_API_BASE_URL`
- `NEXT_PUBLIC_FIREBASE_API_KEY`
- `NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN`
- `NEXT_PUBLIC_FIREBASE_PROJECT_ID`
- `NEXT_PUBLIC_FIREBASE_APP_ID`
